比特位计数

338. 比特位计数

原题链接

思路

动态规划无疑了

考虑二进制数的规律。

[000,001,010,011,100,101,110,111]，分别对应 [0,1,2,3,4,5,6,7]

  • 4-7 的二进制数既是对 0-3 的二进制数的最高位从0变成1
  • 后面的二进制数都是在之前所有二进制的最高位加一位1。

python

class Solution(object):
    def countBits(self, num):
        """
        :type num: int
        :rtype: List[int]
        """
        List1 = [0]
        while(len(List1)<=num):
            List2 = [i+1 for i in List1]
            List1 = List1+List2

        return List1[:num+1]
